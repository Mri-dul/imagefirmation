Pixel Art Generator

Create images from JSON data with pixel coordinates and color values

JSON Input

Enter JSON data with pixel positions and RGB values. Each pixel should have:

  • x - horizontal position
  • y - vertical position
  • color - RGB array [R, G, B]

JSON Format Example

{
  "pixels": [
   {"x": 0, "y": 0, "color": [255, 0, 0]},
   {"x": 1, "y": 0, "color": [0, 255, 0]},
   {"x": 2, "y": 0, "color": [0, 0, 255]},
   // ... more pixels ...
  ]
}
